# 🌫️ Kathmandu Air Quality Analysis

## 📌 Overview
This project analyzes air quality data in Kathmandu, Nepal, with a focus on **PM2.5 pollution levels**, seasonal trends, and overall environmental patterns. The goal is to derive meaningful insights that can support awareness, research, and policy decisions.

---

## 🎯 Objectives
- Analyze PM2.5 concentration trends over time
- Identify seasonal variations in air pollution
- Perform data cleaning and preprocessing
- Visualize pollution patterns using graphs
- Generate insights from exploratory data analysis (EDA)

---

## 🛠️ Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib / Seaborn

---

## 📊 Key Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- PM2.5 trend analysis
- Seasonal variation insights
- Data visualization (charts and plots)

## ⚙️ Setup Instructions

1. Clone the repository
git clone https://github.com/your-username/kathmandu-air-quality-analysis.git

cd kathmandu-air-quality-analysis


3. Create virtual environment
python -m venv venv


4. Activate environment
Windows:
venv\Scripts\activate
Mac/Linux:
source venv/bin/activate


5. Install dependencies
pip install -r requirements.txt


6. Run the notebook
jupyter notebook
